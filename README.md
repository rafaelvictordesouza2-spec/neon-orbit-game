# Neon Orbit: Survivor

Jogo mobile casual em Phaser 3, pronto para deploy estático na Vercel.

## Rodar localmente

Use qualquer servidor estático:

```bash
npx serve .
```

## Publicar na Vercel

1. Envie todos os arquivos desta pasta para a raiz do repositório.
2. Na Vercel, importe o repositório.
3. Framework Preset: `Other`.
4. Build Command: vazio.
5. Output Directory: `.`.

Cada push na branch principal gera um novo deploy automaticamente.
